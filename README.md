# yts

[![](art/install.png)](https://raw.githubusercontent.com/fcannizzaro/yts/master/yts.user.js)

## Setup
1. Setup a [Firebase Project](https://console.firebase.google.com), Realtime Database
2. Edit `API_KEY` and `DATABASE_URL`
3. Enjoy!
